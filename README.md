# GitHub Profile Finder

Ye ek simple React application hai jo GitHub profiles search aur view karne ke liye banai gayi hai.  
Ye GitHub API se real-time data fetch karti hai aur user ki information dikhati hai jaise naam, avatar, followers, following, public repositories, bio, location, aur company.
